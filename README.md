# Smart_Home_Enengy_Management_System
Smart Home Energy Management System
A full-stack web application designed to monitor, control, and optimize energy usage in smart homes. This system enables users to track real-time power consumption, manage devices remotely, and analyze historical energy data for better efficiency.
🚀 Features
📊 Real-time energy monitoring
🏠 Device-level tracking and control
🔌 Remote switching of appliances
📈 Energy usage analytics and insights
🔐 Secure user authentication (JWT-based)
📅 Historical data visualization
🛠️ Tech Stack
Frontend
React.js
Tailwind CSS / CSS
Axios
Backend
Spring Boot
Spring Security
REST APIs
Database
MySQL

Installation & Setup
1. Clone the Repository
git clone https://github.com/Aishwarya765/Smart_Home_Energy_Management_System_765 cd frontend
2. Backend Setup (Spring Boot)
cd shems
Update application.properties: spring.datasource.url=jdbc:mysql://localhost:3306/energy_db spring.datasource.username=root spring.datasource.password=yourpassword
Run the backend: mvn spring-boot:run
Backend will run on: http://localhost:8080
3. Frontend Setup (React)
cd frontend npm install npm start
Frontend will run on: http://localhost:3000
